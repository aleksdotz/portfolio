Project case

Biertje

Een sociale app-concept met friends, games en recap schermen. Use the toggle to switch between dark and light designs and scroll through the screens with the arrows.

Project context

Biertje is a social app concept for a night out with friends, games and recap screens.

Design Process

Through the use of user validation and user testing, I was able to create a strong app concept through several iterations. The final design is ready for development. I plan to pick this project back up in the future when I have more time and resources.
